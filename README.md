# Global Seller Hub

글로벌 셀링 강의를 정리하고 코스별 강의 페이지를 생성하기 위한 정적 사이트입니다.

## 주요 기능

- 공개용 강의 플랫폼 홈
- 글로벌 셀링 코스 카탈로그
- 입문자 학습 로드맵
- 강의 여러 개 생성
- 강의별 카테고리, 난이도, 프로젝트 유형, 기술 스택 관리
- 섹션과 영상 레슨 추가
- 유튜브 링크, 자료 링크, 영상 길이, 공개 상태 입력
- 섹션/영상 순서 이동
- 브라우저 자동 저장
- JSON 내보내기와 불러오기
- 선택한 코스를 독립 강의 페이지 HTML로 생성

## 사용 방법

`index.html`은 방문자용 공개 사이트입니다.

`builder.html`은 코스 페이지를 만드는 관리자용 도구입니다.

1. 새 강의를 만들고 강의 정보를 입력합니다.
2. 섹션과 영상 레슨을 추가합니다.
3. 페이지 파일명을 입력합니다. 예: `a-course`
4. `강의 페이지 생성` 버튼을 누르면 `a-course.html` 파일이 생성됩니다.

GitHub Pages로 배포하면 별도 서버 없이 웹 주소로 접속할 수 있습니다.
