# YouTube Course Curriculum Builder

유튜브 영상을 강의 플랫폼 형태로 묶기 위한 커리큘럼 생성 도구입니다.

## 주요 기능

- 강의 여러 개 생성
- 강의별 카테고리, 난이도, 프로젝트 유형, 기술 스택 관리
- 섹션과 영상 레슨 추가
- 유튜브 링크, 자료 링크, 영상 길이, 공개 상태 입력
- 섹션/영상 순서 이동
- 브라우저 자동 저장
- JSON 내보내기와 불러오기

## 사용 방법

`index.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.

GitHub Pages로 배포하면 별도 서버 없이 웹 주소로 접속할 수 있습니다.
